<h1>Myntra Fashion Product Dashboard</h1>

<h2>Deskripsi Proyek</h2>
<p>
  Dashboard interaktif ini dibuat menggunakan Microsoft Power BI untuk menganalisis
  dataset produk fashion dari Myntra. Analisis difokuskan pada distribusi produk,
  performa brand, preferensi warna, serta perbandingan harga antar brand.
</p>

<h2>Dataset</h2>
<p>
  Dataset diperoleh dari Kaggle dan berisi informasi produk seperti Product ID,
  Product Name, Product Brand, Gender, Price, Number of Images, Description,
  dan Primary Color.
</p>

<h2>Tools dan Teknologi</h2>
<ul>
  <li>Microsoft Power BI</li>
  <li>Microsoft Excel</li>
</ul>

<h2>Insight Utama</h2>
<ul>
  <li>Mengidentifikasi brand dengan jumlah produk terbanyak.</li>
  <li>Mengetahui distribusi produk berdasarkan gender.</li>
  <li>Menganalisis warna yang paling dominan pada katalog produk.</li>
  <li>Membandingkan rata-rata harga antar brand.</li>
</ul>
